AMV II Revision Exercises

    1. For \(f\) a scalar field in \(\mathbb{R}^3\) and \({\bf v}\) a vector field in \(\mathbb{R}^3\), use index notation to prove that \[{\bf \nabla}\times(f{\bf v} ) \;=\; ({\bf \nabla}f)\times {\bf v} + f{\bf \nabla}\times{\bf v}\]

    2. Compute the double integral \[\int_0^1\int_{-x}^{x+1}6\,x^2y\,dy\,dx\] Now reverse the order of integration and recompute the integral.
      Name the theorem which states that your two answers should agree.

    3. Find the generalised derivative of (i) \(|x|\) and (ii) \(e^x\delta(x)\) (full work out required).

    4. Consider the distribution \[\delta^{[N]}_{{\rm sum}}:=\sum_{n=-N}^N \delta_{2\pi n}\] for some \(N \in \mathbb{N}\).

      • Draw a picture representing \(\delta^{[N]}_{{\rm sum}}\) and give a brief explanation of what it might represent.

      • Calculate the Fourier transform of \(\delta^{[N]}_{{\rm sum}}\).

      • Find a simple polynomial solution to the generalised equation \[\delta^{[1]}_{{\rm sum}}(x)\,p(x)=0.\]

    1. [parta] Suppose the level curve defined by \(f(x,y)=c\) can be written as the function \(y=g(x)\), where \(f(x,y)\) is differentiable. Derive an expression for \(dg/dx\) in terms of the partial derivatives of \(f(x,y)\).

    2. Name and state clearly the theorem that guarantees that the level curve from part (a) can indeed be written as a function \(y=g(x)\).

    3. [partc] Consider the function \[f(x,y) \;=\; 2 x^3 + 3 y ^2 + 6 x y\] Determine whether or not the curve \(f(x,y) = c\) can be written in the form \(y=g(x)\), and if not, state clearly the points \((x_0,y_0)\) and corresponding values of \(c\) where problems occur.

    4. Using \(f(x,y)\) as given in part (c), determine whether or not the curve \(f(x,y) = c\) can be written in the form \(x=h(y)\), and if not, state clearly the points \((x_0,y_0)\) where problems occur.

    5. Using \(f(x,y)\) as given in part (c), are there any points where the curve \(f(x,y) = c\) can neither be written as \(y=g(x)\) nor as \(x=h(y)\)?

    1. State Stokes’ theorem, and use it to prove that if \({\bf v}({\bf x})\) is a vector field in \(\mathbb{R}^3\), and \(C\) is a closed curve in \(\mathbb{R}^3\) bounding a surface \(S\) on which \(\nabla\times{\bf v}={\bf 0}\), then \(\oint_C{\bf v}\cdot d{\bf x}=0\).

    2. Now let \({\bf v}({\bf x})\) be given by \[{\bf v}(x,y,z)=\frac{y}{x^2+y^2}\,{\bf e}_1 -\frac{x}{x^2+y^2}\,{\bf e}_2\,.\] Use the result from part 3.1 to evaluate \[I=\oint_C{\bf v}\cdot d{\bf x}\] where \(C\) is the curve \[t\mapsto {\bf x}(t) = r(t)\cos(t)\,{\bf e}_1 + r(t)\sin(t)\,{\bf e}_2 +{\bf e}_3\,\qquad 0\le t\le 2\pi\] and \(r(t)\) is any continuous (and continously-differentiable) function satisfiying \(r(t)>0\) for \(0\le t\le 2\pi\) and \(r(0)=r(2\pi)\). (Hint: evaluate the integral for \(r(t)=1\), and then consider the difference between this and the general case.)

    3. State the divergence theorem.
      Let \(V\) be the part of the region between the paraboloid \(z=x^2+y^2\) and the plane \(z=3\) with positive \(x\) and \(y\) coordinates, that is the set of points \[V=\{(x,y,z):~x\ge 0,y\ge 0, x^2+y^2\le z\le 3\}\,.\] Find the volume of \(V\), and then use the divergence theorem to compute \[\int_{S_c}\mathbf{F}\cdot d\mathbf{A}\] where \(\mathbf{F}(x,y,z) = x\,\mathbf{e}_1 + (2y{+}2)\,\mathbf{e}_2 + 3z\,\mathbf{e}_3\) and \(S_c\) is the curved surface of \(V\), that is the set of points \[S_c=\{(x,y,z):~z=x^2+y^2, x\ge 0,y\ge 0,z\le 3\}\,.\]

    1. Consider the following IVP for \(1<t<100\) and \(f\) a smooth function of \(t\), \[-tx^{\prime\prime}+t^4x^\prime=f(t),\qquad x(1)=0, \,x^\prime(1)=0.\]

      1. Rewrite the ODE above as \(Lx=f(t)\). Is \(L\) formally self-adjoint? Justify your answer.

      2. If the answer to part (a) is yes, is the IVP self-adjoint? If the answer to part (a) is no, transform the operator \(L\) into a Sturm-Liouville operator \({\cal L}\) and then decide whether the new IVP \({\cal L}x=g(x)\) for the modified source term \(g\) and the same initial conditions \(x(1)=0, \,x^\prime(1)=0\) is self-adjoint or not. Justify all your steps.

    2. Suppose you are given four functions, each depending on time \(x^0=t\) and on three space coordinates \(x^i, i\in \{1,2,3\}\), which are the components of the vector \({\bf x}\). Denote these functions by \(h^{0}(x^0, {\bf x}), h^{i}(x^0, {\bf x})\) and suppose that they obey the constraint \[\frac{\partial}{\partial x^0}\,h^{0}(x^0, {\bf x})+\sum_{i=1}^3\,\frac{\partial}{\partial x^i}\,h^{i}(x^0,{\bf x})=0.\]

      1. Calculate the time Fourier transform of this constraint.

      2. Each function \(h^{i}(x^0, {\bf x})\) satisfies a four-dimensional wave equation \[\Box\, h^{i}(x^0, {\bf x})=-T^{i}(x^0,{\bf x}),\] where \(\Box\) is the d’Alembertian operator and \((-T^i)\) is a source term. Given appropriate initial conditions, the solution may be written as \[h^{i}(x^0, {\bf x})=\int_{\mathbb{R}^4}\,\frac{1}{|{\bf x}-\boldsymbol{\xi}|}\,\delta(\,|{\bf x}-\boldsymbol{\xi}|-(x^0-\xi^0)\,)\,\Theta(x^0-\xi^0)\,T^{i}(\xi^0, \boldsymbol{\xi})\,d\boldsymbol{\xi}\,d\xi^0.\]

        • Why can one replace \(\delta(\,|{\bf x}-\boldsymbol{\xi}|-(x^0-\xi^0)\,)\,\Theta(x^0-\xi^0)\) by just \(\delta(\,|{\bf x}-\boldsymbol{\xi}|-(x^0-\xi^0)\,)\) in the integral above?

        • Rewrite \(h^{i}(x^0, {\bf x})\) as a 3-dimensional integral, using part (a) and potentially other properties you should quote.

    1. Find the general solution of the Cauchy-Euler equation \[x^2u^{\prime\prime}(x)+3xu^\prime(x)+(1+\alpha)u(x)=0\] where \(\alpha\) is a given positive constant.

    3. Obtain the eigenfunction expansion for the solution of the inhomogeneous problem \[(x^3u^\prime(x))^\prime=x,\qquad u(1)=0,\,u(e)=0.\] Give the coefficients explicitly, i.e. compute the integrals.
