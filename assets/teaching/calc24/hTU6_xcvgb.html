Calculus I, Tutorial Problem Sheet, Week 6

 

The fundamental theorem of calculus

  1. Let \(F(x)=\int_{\pi}^x t\sin t\, dt.\)  Calculate \(F(\pi),\ F'(x)\) and \(F'(\pi/2).\)

  2. Let \[F(x)=-\int_0^{x^2}\frac{2}{3+e^t}\,dt.\] Find all critical points of \(F(x)\) and determine whether they are local minima, maxima or points of inflection. Prove that \(F(300)>F(310).\)

  3. Calculate the derivative of \(F(x)\) where

    \[\mbox{(a)\ } F(x)=\int_0^{x^3} t\,\cos t\, dt, \quad\quad\quad\quad \mbox{(b)\ } F(x)=\int_{x^2}^1 (t-\sin^2t)\, dt, \quad\quad\] \[\mbox{(c)\ } F(x)=\int_1^{\cos x} \sqrt{1-t^2}\, dt, \quad\quad \mbox{(d)\ } F(x)=\int_{0}^{\sqrt{x}} \frac{t^2}{1+t^4}\, dt \quad\quad\]

  4. Calculate the derivative of the following functions \[\mbox{(a)\ } G(t)=\int_{t^2}^{t^4} \sqrt{u}\, du, \quad\quad \mbox{(b)\ } H(x)=\int_{2x}^{\sin(2x)} 3t^2\, dt, \quad\quad \mbox{(c)\ } y(x)=\int_{-x^2}^{\sqrt{x}} t^2\, dt \quad\quad\]

  5. The function \(f(t)\) is defined in terms of a constant \(k\) by \[f(t)=\begin{cases} \ k & \text{if\, } -1\le t\le 0\\ \ 2k & \text{if\, } 0<t\le 1\\ \ 0 & \text{if\, } |t|>1\\ \end{cases}\] Calculate \(F(x)=\int_{-1}^x f(t)\,dt\) at \(x=-2,\ x=-\frac{1}{2},\ x=\frac{1}{2},\ \) and for all \(x>1.\)

  6. Sketch the graph of the function \[f(x)=\begin{cases} x^2+x & \text{if \ } 0\le x\le 1\\ 2x & \text{if \ } 1<x\le 3.\\ \end{cases}\] Calculate the function \(F(x)=\int_0^x f(t)\,dt,\) with \(\mbox{Dom\, }F=[0,3],\) and sketch its graph. Examine the continuity and differentiability properties of both \(f(x)\) and \(F(x)\) at \(x=1.\)

 

Integration using a recurrence relation or even and odd decomposition

  1. For integer \(n\ge 0\) define \[F_n(x)=\int\frac{x^{n+2}}{\sqrt{x^3+1}}\,dx.\] Find a recurrence relation between \(F_n(x)\) and \(F_{n-3}(x)\) and hence calculate \(F_3(x)\) and \(F_6(x).\)

  2. For integer \(n\ge 0\) define \[I_n=\int_0^{\pi/4}\cos^{n+1}x\,dx.\] Find a recurrence relation between \(I_n\) and \(I_{n-2}\) and hence evaluate \(I_2\) and \(I_4.\)

  3. Evaluate the following integrals \[\mbox{(a)} \ \int_{-\pi/4}^{\pi/4}\frac{e^x\sec^2x}{\cosh x}\,dx,\quad \mbox{(b)} \ \int_{-\pi/3}^{\pi/3}\frac{(1+x)^4\cos x}{1+6x^2+x^4}\,dx,\quad \mbox{(c)} \ \int_{-1}^{1}\frac{x^2}{1+e^{\tan x}}\,dx.\quad\]
