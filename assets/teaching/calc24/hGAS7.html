Calculus I, Gradescope Assignment, Week 7

  1. Find an expression for \(\frac{dy}{dx}\) in terms of \(x\) and \(y\) given the relation \(2y^{3/2}+xy-x=0.\)

    Solution:

     

    Differentiating with respect to \(x\) gives \(3\sqrt{y}y'+y+xy'-1=0\)

    hence \(y'=(1-y)/(3\sqrt{y}+x).\)

  2. Given that \(y\) is a differentiable function of \(x\) and satisfies the equation, \(x^3-2y^2+xy=0\), calculate \(y'\) at the point \((x,y)=(1,1)\).

    Solution:

     

    Differentiating with respect to \(x\) gives \(3x^2-4yy'+y+xy'=0\)

    At \((x,y)=(1,1)\) this becomes \(3-4y'+1+y'=0\)

    so \(y'=4/3.\)

  3. Find all local maxima and minima, and hence the global extreme values, of

    \(f(x)=x^4-2x^2+1\) in the interval \([-2,2].\)

    Solution:

     

    Note that \(f(x)=x^4-2x^2+1\) is twice differentiable at all points.

    \(f'(x)=4x(x^2-1),\) hence \(f'(x)=0\) at \(x=0,\pm 1,\) which are all in \([-2,2].\)

    Also, \(f''(x)=4(3x^2-1).\)

    \(f''(0)=-4<0,\) hence \(x=0\) is a local maximum with \(f(0)=1.\)

    \(f''(\pm 1)=8>0,\) hence \(x=\pm 1\) are local minima with \(f(\pm 1)=0.\)

    \(f'(-2)=-24<0,\) hence \(x=-2\) is an endpoint maximum with \(f(-2)=9.\)

    \(f'(2)=24>0,\) hence \(x=2\) is an endpoint maximum with \(f(2)=9.\)

    The global minimum is therefore \(0\)

    and the global maximum is \(9.\)

  4. Find the global extreme values of \(f(x)=x|x^2-6|-\frac{3}{2}x^2+2\) in \([-2,4].\)

    Solution:

     

    Note that \(f(x)=x|x^2-6|-\frac{3}{2}x^2+2\) is differentiable everywhere in \((-2,4)\) except at \(x=\sqrt{6}.\)

    First restrict to \(x\in(-2,\sqrt{6}),\) then \(f(x)=-x(x^2-6)-\frac{3}{2}x^2+2\) with

    \(f'(x)=-3x^2+6-3x=-3(x+2)(x-1).\)

    Hence \(f'(x)=0\) in \((-2,\sqrt{6})\) iff \(x=1.\)  \(f(1)=\frac{11}{2}.\)

    Now restrict to \(x\in(\sqrt{6},4),\) then \(f(x)=x(x^2-6)-\frac{3}{2}x^2+2\) with

    \(f'(x)=3x^2-6-3x=3(x-2)(x+1).\)

    Hence \(f'(x)\ne 0\) in \((\sqrt{6},4).\)

    The only other possibilities for the global extreme values are \(x=\sqrt{6}\) (where \(f(x)\) is not differentiable) and the endpoints \(x=-2,4.\) Checking these values gives

    \(f(\sqrt{6})=-7\)

    \(f(-2)=-8\)

    \(f(4)=18.\)

    The global maximum is \(\mbox{max}\{\frac{11}{2},-7,-8,18\}=18.\)

    The global minimum is \(\mbox{min}\{\frac{11}{2},-7,-8,18\}=-8.\)

  5. Either find the global maximum or justify that it does not exist for \(f(x)=x^4-2x^2\) in \([-\frac{1}{3},\frac{4}{3}],\)

    Solution:

     

    \(f'(x)=4x(x^2-1)=0\) in \((-\frac{1}{3},\frac{4}{3})\) iff \(x=0,1.\)

    \(f(0)=0\)

    \(f(1)=-1\)

    \(f(-\frac{1}{3})=-\frac{17}{81}\)

    \(f(\frac{4}{3})=-\frac{32}{81}\)

    Global maximum is \(0.\)

  6. Either find the global maximum or justify that it does not exist for \(f(x)=x^4-2x^2\) in \([-\frac{1}{3},2].\)

    Solution:

     

    Same as in the previous question, \(f'(x)=4x(x^2-1)=0\) in \((-\frac{1}{3},2)\) iff \(x=0,1.\)

    \(f(0)=0,\ f(1)=-1,\ f(-\frac{1}{3})=-\frac{17}{81}\)

    except that now \(f(2)=8\)

    hence the global maximum is \(8.\)

  7. Either find the global maximum or justify that it does not exist for \(f(x)=x^4-2x^2\) in \((0,1].\)

    Solution:

     

    \(f'(x)=4x(x^2-1)<0\) in \((0,1)\)

    so \(f(x)\) is decreasing in this interval.

    As the left-hand point is not contained in the given interval \((0,1]\)

    then there is no global maximum in \((0,1].\)
