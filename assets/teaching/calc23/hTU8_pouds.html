First order differential equations

  1. Solve the following differential equations for \(y(x)\)

    (a) \(2xyy'=1+y^2\),

    (b) \((\frac{y}{x}+6x)dx+(\log x-2)dy=0\),

    (c) \((xy+xy^3)dy=\log x\,dx\),

    (d) \((x+y)y'=(x-y)\),

    (e) \(y'=(x^3-2y)/x\),

    (f) \(4xyy'=x^2+3y^2\),

    (g) \(y'\sqrt{2xy}=1\),

    (h) \(2y'+y(ye^{x/2}+1)=0.\)

  2. Solve the following initial value problems for \(y(x)\)

    (a) \(y'+2xy=x,\) with \(y(0)=1,\)

    (b) \((e^{2y}-y\cos(xy))dx+(2xe^{2y}-x\cos(xy)+2y)dy=0,\) with \(y(0)=1,\)

    (c) \(xy'\log x+y=\log x,\) with \(y(e)=1.\)

  3. Solve the following initial value problems for \(x(t)\)

    (a) \(\dot x=4x,\) with \(x(0)=2,\)

    (b) \(4x\dot x=1,\) with \(x(0)=1,\)
