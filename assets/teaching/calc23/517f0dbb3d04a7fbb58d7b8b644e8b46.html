Calculus I, Tutorial Problem Sheet, Week 9

 

Second order differential equations

  1. Solve the following boundary value problems for \(y(x)\)

    (a) \(y''-5y'+6y=0,\) \(y(0)=1, \ y(1/2)=e.\)

    (b) \(y''-2y'+2y=0,\) \(y(0)=3, \ y(\pi/2)=0.\)

    Solution:

     

    (a)  \((Char)\ \lambda^2-5\lambda+6=0=(\lambda-2)(\lambda-3), \quad \lambda=2,3, \quad y=Ae^{2x}+Be^{3x}, \quad\)

    \(y(0)=1=A+B,\quad y(1/2)=e=Ae+Be^{3/2},\) so \(1=A+B\sqrt{e},\)

    giving \(B=0, \ A=1, \quad y=e^{2x}.\)
    (b)  \((Char)\ \lambda^2-2\lambda+2=0, \quad \lambda=1\pm i, \quad y=e^x(A\cos x+B\sin x), \quad\)

    \(y(0)=3=A,\quad y(\pi/2)=0=e^{\pi/2}B,\) so \(B=0,\) giving \(y=3e^{x}\cos x.\)

  2. Solve the following initial value problems for \(y(x)\)

    (a) \(y''-3y'+2y=\cos x,\) \(y(0)=0, \ y'(0)=-1,\)

    (b) \(y''+2y'-3y=e^{-x}+e^{2x}\cos x,\) \(y(0)=1, \ y'(0)=1,\)

    (c) \(y''-5y'+4y=3e^{4x},\) \(y(0)=0, \ y'(0)=-1,\)

    (d) \(y''+4y'+4y=-2e^{-2x}+32e^{2x},\) \(y(0)=-1, \ y'(0)=2.\)

    Solution:

     

    (a) CF: \((Char)\ \lambda^2-3\lambda+2=0=(\lambda-1)(\lambda-2), \quad \lambda=1,2, \quad y_{CF}=Ae^x+Be^{2x}.\)

    PI: \(y=c\cos x+d\sin x, \quad -c\cos x-d\sin x-3(-c\sin x+d\cos x)+2(c\cos x+d\sin x)=\cos x,\)

    \(c-3d=1, \quad d=-3c, \quad c=\frac{1}{10}, \ d=-\frac{3}{10}, \quad y_{PI}=\frac{1}{10}\cos x-\frac{3}{10}\sin x.\)

    \(y=Ae^x+Be^{2x}+\frac{1}{10}\cos x-\frac{3}{10}\sin x, \quad y(0)=0=A+B+\frac{1}{10}, \quad\)

    \(y'=Ae^x+2Be^{2x}-\frac{1}{10}\sin x-\frac{3}{10}\cos x, \quad y'(0)=-1=A+2B-\frac{3}{10}, \quad A=\frac{1}{2},\ B=-\frac{3}{5},\)

    \(y=\frac{1}{2}e^x-\frac{3}{5}e^{2x}+\frac{1}{10}\cos x-\frac{3}{10}\sin x.\)
    (b) CF: \((Char)\ \lambda^2+2\lambda-3=0=(\lambda-1)(\lambda+3), \quad \lambda=1,-3, \quad y_{CF}=Ae^x+Be^{-3x}.\)

    PI: \(y=ae^{-x}+e^{2x}(b\cos x+c\sin x),\) gives

    \(ae^{-x}+e^{2x}((3b+4c)\cos x+(3c-4b)\sin x) -2ae^{-x}+e^{2x}((4b+2c)\cos x+(4c-2b)\sin x)\)

    \(-3ae^{-x}+e^{2x}(-3b\cos x-3c\sin x)=e^{-x}+e^{2x}\cos x,\)

    \(a=-\frac{1}{4}, \ 6c+4b=1, \ 2c=3b, \ b=\frac{1}{13}, \ c=\frac{3}{26}, \quad y_{PI}=-\frac{1}{4}e^{-x}+e^{2x}(\frac{1}{13}\cos x+\frac{3}{26}\sin x),\)

    \(y=Ae^x+Be^{-3x}-\frac{1}{4}e^{-x}+e^{2x}(\frac{1}{13}\cos x+\frac{3}{26}\sin x),\)

    \(y(0)=1=A+B-\frac{9}{52},\quad y'(0)=1=A-3B+\frac{27}{52}, \quad A=1, \ B=\frac{9}{52},\)

    \(y=e^x+\frac{9}{52}e^{-3x} -\frac{1}{4}e^{-x}+e^{2x}(\frac{1}{13}\cos x+\frac{3}{26}\sin x).\)
    (c) CF: \((Char)\ \lambda^2-5\lambda+4=0=(\lambda-1)(\lambda-4), \quad \lambda=1,4, \quad y_{CF}=Ae^x+Be^{4x}.\)

    PI: \(y=axe^{4x}, \quad ae^{4x}(16x+8-5(4x+1)+4x)=3e^{4x}, \quad a=1, \quad y_{PI}=xe^{4x}.\)

    \(y=Ae^x+Be^{4x}+xe^{4x}, \quad y(0)=0=A+B, \quad y'(0)=-1=A+4B+1, \quad A=\frac{2}{3},\ B=-\frac{2}{3},\)

    \(y=\frac{2}{3}e^x-\frac{2}{3}e^{4x}+xe^{4x}.\)
    (d) CF: \((Char)\ \lambda^2+4\lambda+4=0=(\lambda+2)^2, \quad \lambda=-2, \quad y_{CF}=e^{-2x}(Ax+B).\)

    PI: \(y=ax^2e^{-2x}+be^{2x}, \quad ae^{-2x}(4x^2-8x+2-8x^2+8x+4x^2)+16be^{2x}=-2e^{-2x}+32e^{2x},\)

    \(a=-1, \ b=2, \quad y_{PI}=-x^2e^{-2x}+2e^{2x}.\)

    \(y=e^{-2x}(Ax+B)-x^2e^{-2x}+2e^{2x}, \quad\)

    \(y(0)=-1=B+2, \ y'(0)=2=A-2B+4, \ A=-8, B=-3.\)

    \(y=e^{-2x}(-8x-3)-x^2e^{-2x}+2e^{2x}.\)

  3. Solve \(y''+y=\tan x.\)

    Solution:

     

    CF: (CHAR) \(\lambda^2+1=0\), so \(\lambda=\pm i\) giving \(y_{CF}=A\cos x+B\sin x.\)

    For PI use the method of variation of parameters with \(y_1=\cos x\) and \(y_2=\sin x.\)

    \(W(y_1,y_2)=y_1y_2'-y_2y_1'=\cos^2x+\sin^2x=1.\)

    \(u_1=-\int \tan x \sin x\, dx=-\int \frac{\sin^2x}{\cos x}\,dx =\int \frac{\cos^2x-1}{\cos x}\,dx = \int \cos x -\sec x \,dx\)

    \(=\sin x -\int \sec x \frac{(\sec x+\tan x)}{(\sec x+\tan x)}\,dx =\sin x-\int \frac{\sec^2 x+\sec x\tan x}{\sec x+\tan x}\,dx =\sin x-\log(\sec x+\tan x)\)

    \(u_2=\int \tan x \cos x\,dx=\int \sin x\,dx = -\cos x\)

    \(y_{PI}=\cos x(\sin x-\log(\sec x+\tan x))-\sin x\cos x =-\cos x\,\log(\sec x+\tan x)\)

    \(y=y_{CF}+y_{PI}=A\cos x+B\sin x -\cos x\,\log(\sec x+\tan x)\)

  4. Solve \(y''-2y'+y=e^x\log x.\)

    Solution:

     

    CF: (CHAR) \(\lambda^2-2\lambda +1=0=(\lambda-1)^2\) giving \(\lambda=1\) repeated \(y_{CF}=e^x(A+Bx)\)

    For PI use the method of variation of parameters with \(y_1=e^x\) and \(y_2=xe^x\)

    \(W(y_1,y_2)=y_1y_2'-y_2y_1'=e^{2x}.\)

    \(u_1=-\int xe^x \bigg(\frac{e^x\log x}{e^{2x}}\bigg)\,dx=-\int (\log x)x\,dx =-(\log x)\frac{x^2}{2}+\int \frac{1}{x}\frac{x^2}{2}\,dx\)

    \(=-\frac{1}{2}(\log x){x^2}+\frac{1}{4}x^2\)

    \(u_2=\int e^x \bigg(\frac{e^x\log x}{e^{2x}}\bigg)\,dx= \int (\log x)\,dx=(\log x)x-\int \frac{1}{x} x\,dx= (\log x)x-x.\)

    \(y_{PI}=e^x\bigg(-\frac{1}{2}(\log x){x^2}+\frac{1}{4}x^2\bigg) +xe^x((\log x)x-x) =x^2e^x(\frac{1}{2}\log x-\frac{3}{4})\)

    \(y=y_{CF}+y_{PI}=e^x(A+Bx) +x^2e^x(\frac{1}{2}\log x-\frac{3}{4})\)

  5. Solve the following pairs of first order differential equations for \(y(x),z(x)\) by first finding a second order differential equation for \(y(x)\)

    (a) \(y'=y+3z, \quad \ z'=-3y+z,\)

    (b) \(y'+6y-2z=e^x, \quad \ z'+8y-4z=2e^x.\)

    Solution:

     

    (a)

     \(z=\frac{1}{3}(y'-y), \quad \frac{1}{3}(y''-y')=-3y+\frac{1}{3}(y'-y),\)

    \(y''-2y'+10y=0,\)

    \((Char), \ \lambda^2-2\lambda+10=0,\quad \lambda=1\pm 3i,\)

    \(y=e^x(A\cos(3x)+B\sin(3x)).\)

    \(z=\frac{1}{3}(y'-y)=e^x(B\cos(3x)-A\sin(3x)).\)
    (b)

     \(z=\frac{1}{2}(y'+6y-e^x), \quad \frac{1}{2}(y''+6y'-e^x)+8y-2(y'+6y-e^x)=2e^x,\)

    \(y''+2y'-8y=e^x,\)

    \((Char), \ \lambda^2+2\lambda-8=0=(\lambda-2)(\lambda+4),\quad \lambda=2,-4,\)

    \(y_{CF}=Ae^{2x}+Be^{-4x}.\)

    \(y_{PI}=ae^{x}\) gives \(-5ae^x=e^x\) hence \(a=-\frac{1}{5}\) so \(y_{PI}=-\frac{1}{5}e^x.\)

    \(y=Ae^{2x}+Be^{-4x}-\frac{1}{5}e^x.\)

    \(z=\frac{1}{2}(y'+6y-e^x)=4Ae^{2x}+Be^{-4x}-\frac{6}{5}e^x.\)
